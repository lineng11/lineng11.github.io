Django安装

Django自称是“能够很好地应对应用上线的Web框架”，在开发之前，有必须安装必要的组件。这里包括依赖组件和Django本身。

安装

用python pip 安装Django

pip install django==1.8.2

安装成功如下。

  • 检查Django是否安装成功

卸载Django

pip uninsstall django

pip命名异常解决

虽然安装过程中选择了pip，但是在命令行输入pip后仍然不能成功，尝试一下方法，终于解决问题:

  • 使用下面的语句确保没有报错了

python -m ensurepip

  • 使用pip -V命令查

    pip -V 或者 pip --version

  • 可能是环境没配置好，所以更新一下pip

python -m pip install --upgrade pip

创建一个Django

命令：

django-admin.py startproject XXX(工程名)

  • 如果出现以下问题：

python: can’t open file ‘django-admin.py’: [Errno 2] No such file or directory

如果没猜错你是在 env虚拟环境 下创建的工程

  • 确保django是否安装正确

    conda list

    如果没有？ django

    那么请执行安装虚拟环境下的django

    conda install django

启动Django

python manage.py runserver

在pyCharm运行配置：

  • 注意 File | Settings | Project: new_project | Project Interpreter 添加env虚拟的python，确保有Django

  • 运行 run设置 Parameters : runserver

创建App应用

python manage.py startapp Blog(业务名)
