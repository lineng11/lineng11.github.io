Django方式

goods/views.py

以上三种Django方式获取GoodsListView

  • 方法一太过繁琐，Modles需要 one to one 取出，并且字段不能出错

  • 方法二 可以用 model_to_dict 函数转换成元组形式，缺点是某些特殊的字段(ImageFieldFile 和add_time)不能自动Serializers序列化

  • 方法三 通过Serializers.序列化转换成’json’相对而言已经是不错的方式，但是Model的扩展性不够强，image地址不是全路径
