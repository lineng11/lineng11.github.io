之前自己搭建的VPN，其系统为Centos6，刚好想把python项目部署上去，记录方便你我。

centos6 自带有python，只不过是 python2 版本的，如果你想安装个python3怎么办呢？

因为 yum 源中并没有现成的 python3 程序，所以必须要自己手动编译安装。

那么就安装以下步骤来吧！

python位置

首先了解python2的存放的目录在哪里？

由此可以知晓python在 /usr/bin 目录中

安装依赖包

安装静态库

# yum install -y openssl-static

注：如果不安装该静态库，会导致python的pip安装失败

安装gcc

# yum install -y gcc wget

# yum groupinstall "Development tools"

# yum install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel

下载和安装Python3

通过官网下载Python3安装包

# wget http://python.org/ftp/python/3.6.1/Python-3.6.1.tar.xz

解压安装包

# tar xf Python-3.6.1.tar.xz
